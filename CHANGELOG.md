# Changelog

All notable changes to Drauniav are documented here.

## 0.4.1 - 2026-04-13

### Added

- Added stereo channel modes for the visualizer.
- Added an optional horizontal visualizer mirror mode for correcting the frequency axis after rotation.
- Added separate persisted file dialog directories for background images, audio files, and output videos.

### Fixed

- Fixed draft release creation when checking for a release that does not exist yet.

## 0.4.0 - 2026-03-06

### Added

- Added visualizer rotation controls.

### Changed

- Improved preview overlay orientation marker visuals.
- Stabilized overlay interaction behavior.

## 0.3.0 - 2026-03-06

### Added

- Added reproducible portable release tooling and documentation.

### Changed

- Grouped experimental showfreqs controls into an advanced section and defaulted them off.

## 0.2.0 - 2026-03-03

### Added

- Added video options dialog.

### Changed

- Stabilized output and export behavior.

## 0.1.0 - 2026-03-02

### Added

- Added README, screenshot asset, logo pipeline, application/about icon handling, FFmpeg missing dialog, MIT license, i18n, menu, and about dialog.
- Added color picker support and eyedropper lens.
- Added the first working FFmpeg-based image/audio visualizer workflow.

### Changed

- Renamed the application from AudioVisualizer to Drauniav.
- Moved CLI output into the application window.
- Improved area positioning behavior.
